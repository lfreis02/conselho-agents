# Referência: Análise e estatística honesta

Objetivo: tirar conclusão que se sustenta. A estatística do dia a dia é
pouca coisa bem usada; o valor está em não se enganar. Um número sem
comparação e sem contexto não é uma conclusão.

## Princípios
- Todo número precisa de referência: comparado com quê? Antes? Outro
  grupo? Meta? "Vendemos 500" não diz nada sozinho.
- Prefira o resumo que representa o dado, não o que soa melhor.
- Amostra pequena ou período curto = incerteza alta = ressalva
  explícita. Não maquie variação de acaso como tendência.

## Ferramentas mínimas, bem usadas
1. Média x mediana: com valores extremos (salários, preços, tempo de
   resposta) a média engana; a mediana representa melhor o caso típico.
   Mostre as duas quando divergirem muito.
2. Porcentagem: deixe claro "% de quê". Variação percentual de base
   pequena engana (de 2 para 3 é "+50%"). Diferença de pontos
   percentuais não é o mesmo que variação percentual.
3. Crescimento: distinga absoluto de relativo, e uma vez de recorrente.
   Compare períodos comparáveis (mesmo mês do ano, por sazonalidade).
4. Distribuição antes de média: olhe a cara do dado (mín, máx, como se
   espalha). Duas médias iguais podem esconder realidades opostas.
5. Comparar grupos: só compara justo se os grupos forem comparáveis
   (mesmo período, mesma definição, tamanhos parecidos).

## Armadilhas de raciocínio (nomeie quando aparecerem)
- Correlação não é causa: duas coisas subindo juntas podem ter uma
  terceira causa comum, ou ser coincidência. Nunca afirme causa a
  partir de correlação.
- Amostra viesada: quem respondeu não representa todos (só clientes
  satisfeitos, só quem abriu o e-mail). A conclusão vale só para quem
  entrou na amostra.
- Cherry-picking: escolher o recorte, período ou métrica que confirma a
  tese e ignorar o resto. Pergunte-se o que foi deixado de fora.
- Confundir sinal com ruído: variação pequena entre poucos casos
  costuma ser acaso, não descoberta.

## Bandeiras vermelhas / limites
- Pedido para "provar" uma conclusão já decidida: recuse o cherry-
  picking e mostre o quadro completo, mesmo contrariando a expectativa.
- Amostra minúscula ou dado enviesado: responda com a ressalva na
  frente, não escondida no rodapé.
- Nada de p-valor, teste ou modelo inventado sem o dado permitir, e
  nada de benchmark de mercado que a pessoa não trouxe.
