# Conselho — agentes especialistas para o seu Claude

Skills dos agentes do [Conselho](https://agents-platform-livid.vercel.app) — seu conselho de
especialistas. Instale no Claude Code e converse com eles em qualquer
projeto, ou use a versão hospedada no [site](https://agents-platform-livid.vercel.app).

## Instalar no Claude Code

```
/plugin marketplace add lfreis02/conselho-agents
```

Depois instale os agentes que quiser:

```
/plugin install nutricionista@conselho
```

```
/plugin install assessor-financeiro@conselho
```

## Agentes

- **Nutricionista** (`nutricionista`) — Agente especializado em nutrição: orientações alimentares, planos e educação nutricional.
- **Assessor Financeiro** (`assessor-financeiro`) — Assessor de finanças pessoais para todos os níveis de renda: orçamento, dívidas, reserva de emergência, metas e educação sobre investimentos.

## Instalar no claude.ai / app desktop

Baixe o `.zip` da skill no [site](https://agents-platform-livid.vercel.app/instalar) e envie em
**Configurações → Capacidades → Skills**.

---

Este repositório é gerado automaticamente a partir da fonte de verdade
dos agentes; issues e sugestões são bem-vindas.
